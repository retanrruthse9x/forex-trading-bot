## FAQs

**Q: How do I configure my Forex trading platform with the bot?**  
A: You need to generate API keys from your Forex platform (e.g., MetaTrader) and input them into the `api_keys.json` configuration file. The bot will use these keys to interact with your account.

**Q: Can I customize the trading strategies?**  
A: Yes, the bot allows you to choose from pre-built strategies or create your own using the strategy manager. You can define parameters like risk limits, target assets, and trade volume.

**Q: Is this bot compatible with all Forex platforms?**  
A: The bot currently supports MetaTrader 5 (MT5) but can be extended to support other platforms with minor modifications to the API interaction code.
